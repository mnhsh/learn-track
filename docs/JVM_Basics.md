## What is JDK?

JDK (Java Development Kit) provides us the tools make java applications

**It includes:**
- JVM
- JRE
- Java compiler (javac)
- Other development tools

## What is JRE?

JRE (Java Runtime Environment) provides the runtime environment for java applications

**It includes:**
- JVM
- Core libraries

## What is JVM?

JVM (Java Virtual Machine) executes the bytecode produced after compiling the java code

## What is Bytecode?

When a Java file is compiled it produces bytecode which can be executed by JVM

## Write Once, Run Anywhere

A Java program is compiled into bytecode only once. The same bytecode can run on any operating system as long as they have a compatible JVM installed.

This makes Java portable across Windows, Linux, and macOS.
